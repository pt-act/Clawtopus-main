> "Eight arms, infinite memory."

## The Problem

Every AI assistant suffers from the same issue: **amnesia**. Start a new conversation, and it's like meeting for the first time. You repeat the same context, re-explain your preferences, and rebuild from scratch.

Cloud AI services like ChatGPT and Claude have limited memory that:

- Expires after a few months
- Is stored on someone else's servers
- Can't be searched or organized
